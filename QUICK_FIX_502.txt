┌─────────────────────────────────────────────────────────────────────┐
│  🔍 WHAT TO LOOK FOR IN LOGS:                                       │
└─────────────────────────────────────────────────────────────────────┘

    ❌ "could not connect to server"
       → DATABASE NOT CONNECTED

    ❌ "KeyError: 'SECRET_KEY'"
       → MISSING SECRET_KEY VARIABLE

    ❌ "ModuleNotFoundError"
       → BUILD FAILED

    ❌ "Migration failed"
       → DATABASE SCHEMA ISSUES

    ❌ "Failed to bind"
       → PORT CONFIGURATION ISSUE


┌─────────────────────────────────────────────────────────────────────┐
│  ⚡ MOST LIKELY FIXES (Try these first):                            │
└─────────────────────────────────────────────────────────────────────┘

    1️⃣  MISSING SECRET_KEY
        ───────────────────────────────────────────────────
        → Go to: Render Dashboard → Your Service → Environment
        → Click "Add Environment Variable"
        → Key: SECRET_KEY
        → Value: ism+7l(h49h2v4(p9jg_9vlu)-m89nc_&pfztmavk_m4$6iwq*
        → Save Changes (auto-redeploys)


    2️⃣  DATABASE NOT LINKED
        ───────────────────────────────────────────────────
        → Go to: Render Dashboard → Your Service
        → Scroll to "Environment Variables"
        → Check if DATABASE_URL exists
        → If missing: Click "Add Database" and link PostgreSQL


    3️⃣  CLEAR CACHE & REDEPLOY
        ───────────────────────────────────────────────────
        → Go to: Render Dashboard → Your Service
        → Click "Manual Deploy" button
        → Select "Clear build cache & deploy"
        → Wait 5-10 minutes


    4️⃣  RUN MIGRATIONS MANUALLY
        ───────────────────────────────────────────────────
        → Go to: Render Dashboard → Your Service → "Shell" tab
        → Run: python manage.py migrate --noinput
        → Run: python manage.py seed_plans

└─────────────────────────────────────────────────────────────────────┘

        python manage.py healthcheck
